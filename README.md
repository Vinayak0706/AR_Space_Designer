# \# AR Space Designer

# 

# A mobile AR application built with Unity and AR Foundation.

# 

# \## Features

# \- Plane detection

# \- Object placement

# \- Object interaction (move, scale, rotate)

# \- Reset and re-scan environment

# 

# \## Tech

# \- Unity

# \- AR Foundation

# \- ARCore (Android)

# 

# \## How to Run

# \- Open project in Unity

# \- Switch to Android

# \- Build and run on a real device

# 

# > AR features require a physical device.

#
